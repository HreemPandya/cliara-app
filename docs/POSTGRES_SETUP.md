# PostgreSQL Setup Guide for Cliara

## Status

No manual PostgreSQL setup is required. Cliara automatically downloads and configures PostgreSQL on your machine when needed.

## Optional: Use Your Own PostgreSQL

If you want to point Cliara at an existing PostgreSQL instance (local, Docker, or cloud), configure it in `~/.cliara/config.json` with your connection string and credentials. Otherwise, you can ignore PostgreSQL entirely.
